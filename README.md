# Dune

macOS React Native app using Bun.

## Setup

```sh
bun install
cd macos && bundle exec pod install && cd ..
```

## Run

```sh
bun start       
bun run macos      
```

## Scripts

- `bun start` - Start Metro bundler
- `bun run macos` - Build and run app
- `bun test` - Run tests
- `bun run type-check` - Type check
